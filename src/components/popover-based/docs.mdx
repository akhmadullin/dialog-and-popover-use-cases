# Компоненты на основе атрибута popover

На основе атрибута popover можно делать всплывающие элементы, которым не нужна модальность.

Если добавить к popover CSS anchor positioning, то получаются по-настоющему гибкие и адаптивные компоненты, которые могут подстраиваться под различные положения элемента относительно вьюпорта.

## Tooltip

Очень часто в интерфейсах нужно показать тултип - подсказку, которая дает пользователю больше информации о том, что происходит на странице.

### Tooltip, адаптирующийся в рамках одной оси

Иногда нужно, чтобы тултип показывался либо сверху/снизу, либо справа/слева.

## Toast

Тосты – всплывающие нотификации.

Бывают полезны, когда есть необходимость уведомить пользователя о произошедшем событии. 

<Canvas of={ToastStories.NotChangingEachOther}  />

Порой необходимо, чтобы нотификация об одной и том же событии, сменяла своего предшественника. Например, если это нотификации об одной и той же ошибке.
